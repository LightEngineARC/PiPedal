# PiPedal

Turn a Raspberry Pi into a guitar effects pedal

##    Set up instructions for the PiPedal

##### Minimum Hardware

* Raspberry Pi
* Monitor, mouse, and keyboard
* Input Audio Cable
* Output Audio Cable
